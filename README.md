A Crop/Weed Field Image Dataset
=============

The Crop/Weed Field Image Dataset (CWFID) accompanies the following publication: "Sebastian Haug, JÃ¶rn Ostermann: A Crop/Weed Field Image Dataset for the Evaluation of Computer Vision Based Precision Agriculture Tasks, [CVPPP 2014](http://www.plant-phenotyping.org/CVPPP2014) Workshop, [ECCV 2014](http://eccv2014.org)"

This dataset comprises field [images](images), vegetation segmentation [masks](masks) and crop/weed plant type [annotations](annotations). The paper provides details, e.g. on the field setting, acquisition conditions, image and ground truth data format.
